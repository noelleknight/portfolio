My Work

Sequence

Sequence is an application for yogis. It's a tool for teachers and practitioners to create full length yoga sequences. The app generates a series of yoga poses based on preferences set by the user. Users are able to save their sequences while logged in so they can use them repeatedly. This was my final project at the Iron Yard and it's one I'm extremely passionate about as a yogi and developer.

Random Repos

Random Repos is an application that utilizes the github API in a fun way. By entering a repository name you can see a list of all of the developers associated with it, including their avatars and a link to their GH profile. I utilized the jQuery animation method in this project to add some cool effects to a simple application.
